USAFICO

USAFICO (Usafi in congo) is a web/mobile application aimed at improving sanitation services in urban areas. It streamlines communication and coordination between citizens, sanitation companies, and local authorities.

Kongo$ha

Kongo$ha is a mobile application designed to empower users to invest in local projects. The main goal is to promote financial inclusion by providing a simple, secure, and educational platform for managing investments.


Flashgo

FlashGo is a fast and lightweight messaging application designed for real-time communication. The app offers essential chat features with a focus on speed, simplicity, and user privacy. It was developed as a prototype to explore real-time backend technologies and mobile UI/UX best practices.

FutureFarm

FutureFarm is a digital platform aimed at transforming traditional farming through smart technologies. The project provides farmers with tools to monitor crops, manage resources efficiently, and access farming advice. It was built to address challenges like low yields, climate variability, and limited access to agricultural data.
